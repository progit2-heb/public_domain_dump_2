אוֹת אֵל גֵּאֶה

אברהם אבן עזרא

אוֹת אֵל גֵּאֶה / יוֹם יוֹם אֶרְאֶה, / כּוֹנַנְתָּ אֶ-/ רֶץ וַתַּעֲמֹד.

יְיָ מֶלֶךְ, / יְיָ מָלָךְ, / אֲדֹנָי אֱלֹהַי / גָּדַלְתָּ מְאֹד.

בְּקִרְבִּי קְדשִׁי, / אֵיכָה אֶשִׁי / זִכְרוֹ – וְנַפְשִׁי / יוֹדַעַת מְאׂד!

רוּחֲךָ טוֹבָה / תּוֹרֶה נְתִיבָה / לִי, כִּי רְחָבָה / מִצְוָתְךָ מְאֹד. הֵן קָדַמְתָּ, / עַל כֹּל רַמְתָּ, / כּי עָצַמְתָּ / מִמֶּנּוּ מְאׂד.

מַרְאֶה עֵינַי / וּמִשְׁמַע אָזְנַי: / גָּדוֹל אֲדׂנָי / וּמְהֻלָּל מְאׂד!
