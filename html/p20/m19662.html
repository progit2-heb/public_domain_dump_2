אֱלֹהִים חֵילִי יוֹצֵר הָרִים

אברהם אבן עזרא

סליחה (?)

אֱלֹהִים חֵילִי, / יוֹצֵר הָרִים,

שִׁמְעָ[ה] קוֹלִי, / מַשְׁפִּיל וּמֵרִים,

וְחִישׁ [נָא] גוֹאֲלִי, / מַתִּיר אֲסוּרִים,

בִּזְכוּת הוֹרִים / מַחֲמַדֵּי עָיִן.

[….] יִשְׂרָאֵל / […] ל[…..]

וּמְשִׁיחָם שְׁלַח, / רָם [….]

[…..] צֶמַח / בִּזְכוּת [….]

[…..] רְחוּמִים / עַיִן בְּעָיִן.

בָּאתִי לְבֵיתָךְ, / רוֹכֵב עֲרָבוֹת,

יַשֵּׁר עֲדָתָךְ / הוֹלִיכֵם נְתִיבוֹת,

וְהֵם סְגֻלָּתָךְ / דּוֹרְשֵׁי תְשׁוּבוֹת,

[……] הָעָיִן.
