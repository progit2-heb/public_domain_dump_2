מוחרך לברכו

אברהם אבן עזרא

כָּל אֲשֶׁר / יוּכַל לְהִתְרַצּוֹת / אֶל אֵל בְּנָאֳמו –

גַּדְּלוּ / לַיָי אִתִּי / וּנְרוֹמְמָה שְׁמוֹ! אַחֲלֵי / לִבִּי שֶׁיֶּחְזֶה / שֵׁם אֵל בְּרָאוֹ.

בַּעֲלֵי / לֵב בִּקְּשׁוּ רָזֵי / הַשֵּׁם וּפִלְאוֹ,

יַעֲלֶה / עַל נִיב אִישׁ חוֹזֶה, / ‘אֶהְיֶה’ קְרָאוֹ,

וַעֲלֵי / דָּתוֹ נִכְתַּב: זֶה / זִכְרוֹ וְזֶה שְׁמוֹ. בּוֹעֲרִים / אֵיךְ לֹא יַכִּירוּ / שֵׁם צוּר יְלָדָם?

סוֹרֲרִים / אֵיךְ לֹא יַיְשִׁירוּ / אֶת מַעֲבָדָם?

סוֹחֲרִים / בִּזְבוּל יָעִירוּ: / אַיֵּה כְבוֹדָם?

אוֹמְרִים / לִבְנֵי אִישׁ: שִׁירוּ / לוֹ, בָּרְכוּ שְׁמוֹ! רָשְׁמוּ / צוּרוֹת נִפְרָדוּ / חֻקֵּי אֱמוּנוֹת,

עָצְמוּ / עַד לֹא יֻגָּדוּ, / וּבְרֹב תְּבוּנוֹת

חָכְמוּ, / כִּי בָם יֻלָּדוּ / כָּל הַתְּמוּנוֹת –

יִנְאֲמוּ / זֶה לָזֶה: הוֹדוּ / לוֹ, בָּרְכוּ שְׁמוֹ.
