אֵל יְגַלֶּה לְעֵינַי

אברהם אבן עזרא

אֵל יְגַלֶּה לְעֵינַי / מַעֲבָדָיו,

נוֹדְעָה יַד אֲדֹנָי / אֶל עֲבָדָיו.

בּוֹ לְבַדוֹ אַאְמִין; / אֶדְרְשֵׁהוּ,

אֶלְמְדָה מַעְשַׁי מִן / מַעֲשֵׁהוּ.

אַשְׂמְאִיל אוֹ אֵימִין — / אֶפְגְּשֵׁהוּ,

וַעֲלֵי רַעְיוֹנַי / שָׁם פְּקִידָיו,

נִצְּבוּ עַל פָּנַי / סוֹד מְעִידָיו.

רָחֲבָה לֹא תִמַּד / דָּת יְקָרוֹ.

דֹּק וְחוּגוֹ עָמַד / עַל דְּבָרוֹ.

מִשְׁפְּטֵי פִיו לָמַד / עַם בְּחָרוֹ —

בָּא אֲזַי מִסִּינַי / עִם גְּדוּדָיו,

וַעֲדֶנָּה בְאָזְנַי / קוֹל פְּחָדָיו.

מָה אֲבַקֵּשׁ עוֹד לִי? / רַק רְצוֹנוֹ!

כִּי יְסוֹד מִשְׁאָלִי — / הוּא יְבִינוֹ,

וַאֲשֶׁר הוּא טוֹב לִי — / הוּא יְכִינוֹ.

הֵן לְזֹאת הֶגְיוֹנַי / אֶל יְדִידָיו:

אֶהֱבוּ אֶת אֲדֹנָי / כָּל חֲסִידָיו!

מִפְלְאוֹתָיו — כַּמָּה / אִישׁ יְבִינֵם?

מַעֲלָתָם עָצְמָה, / מִי יְשַׁנֵּם?

מִצְעֲדֵי אִישׁ חָכְמָה — / הוּא יְכִינֵם.

יוֹם וְלֵיל הֶגְיוֹנַי / נִיב יְדִידָיו:

אֶהֱבוּ אֶת אֲדֹנָי / כָּל חֲסִידָיו!
