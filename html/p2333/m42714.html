הֵן הַחֹרֶף

אברהם ברוך מני

לבעל חוטם גדול

הֵן הַחֹרֶף / פָּנָה עֹרֶף / וָאֶשְׁאֲלָה / כֹּה בְּבֶהָלָה

אָבִיב זָרַח / דֶּשֶׁא פָרַח / אֵיךְ יָכוֹל הוּא / לְסָבְלֵהוּ

לְעֻמָּתוֹ / וְלִקְרָאתוֹ / אִם בּוֹ אֵיפֹא / עֲלֵי גוּפוֹ

הֵן הַחוּצָה / לוֹ אָרוּצָה. / כִּפְלַיִם לוֹ / בְּמִשְׁקָלוֹ.

שָׁם רָאִיתִי / שָׁם חָזִיתִי / שׁוֹמְעִים מִלַּי / עֲנוּ אֵלַי

אִישׁ אַף גָּדוֹל / כְּמוֹ מִגְדֹּל / עַל כֵּן הוֹלֵךְ / כְּמוֹ הֵלֶךְ

כְּמוֹ מִבְצָר / כְּמוֹ אוֹצָר / דַּל שָׁחוֹחַ / בְּלִי כֹחַ

כְּמוֹ אֹרֶן / וְגַם תֹּרֶן. / כִּי לוֹ אַפּוֹ / הִכְבִּיד אַכְפּוֹ.
