לְשָׁנִים דְּשֵׁנִים

אברהם ברוך מני

ברכה לידיד

לְשָׁנִים דְּשֵׁנִים / וּזְמַנִּים לֹא נִמְנִים / וּזְקוּנִים נְכוֹנִים / בִּרְנָנִים תַּגִּיעוּ

בְּבָנִים נְבוֹנִים / הֲמוֹנִים אֱמוּנִים / רוֹזְנִים וּסְגָנִים / פְּנִינִים יַבִּיעוּ

וְכַמָּה וְכַמָּה / בְּחָכְמָה וּמְזִמָּה / כְּחַמָּה וְכִימָה / בִּנְעִימָה תוֹפִיעוּ

וּבְעָצְמָה בְקוֹמָה / שְׁלֵמָה וּתְמִימָה / כְחוֹמָה רוֹמֵמָה / מֵאֵימָה תַרְגִּיעוּ

כִּנְהָרִים נִגָּרִים / סַפִּירִים וּמְהִירִים / נֶהְדָּרִים נִבְחָרִים / חֲדָרִים תַּצִּיעוּ

בִּדְבָרִים יְקָרִים / אוֹצָרִים שְׁמוּרִים / מִבְצָרִים וּשְׁעָרִים / נִגְזָרִים תּוֹשִׁיעוּ

בִגְבוּרָה יְתֵרָה / בְּתוֹרָה וְסִפְרָא / וּגְמָרָא בְרוּרָה / אוֹרָה תַשְׁפִּיעוּ

נָא אֵל נָא שְׁמַע נָא / תְּחִנָּה לְמָנָה / נְתוּנָה נְתוּנָה / וּרְנָנָה יָרִיעוּ.
