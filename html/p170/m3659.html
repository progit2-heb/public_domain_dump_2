מהיום הבא ומפחד היותו

משה אבן עזרא

[מהיום הבא ומפחד היותו] / משה אבן עזרא

מֵהַיּוֹם הַבָּא / וּמִפַּחַד הֱיוֹתוֹ

כֶּעָשָׁן מֵאֲרֻבָּה / תַעַל מְהוּמָתוֹ,

תְּבוּסָתוֹ רַבָּה – / וּמִי יוּכַל שְׂאֵתוֹ

וּמִי הָעוֹמֵד / בְּהֵרָאוֹתוֹ?

שׁוֹקֵט וּבוֹטֵחַ – / אֵימָה תְבַעֲתֵהוּ,

וּמָוֶת שׁוֹכֵחַ – / שׁוֹאָה תְבוֹאֵהוּ,

וְשׁוֹכֵב שָׂמֵחַ – / צְעָקָה תְעִירֵהוּ,

תֵּצֵא נִשְׁמָתוֹ / יָשׁוּב לְאַדְמָתוֹ.

הוּא הַיּוֹם הַמַּר,/ כַּתַּנּוּר בּוֹעֵר,

כְּמֵי הַיָּם יֵמַר / וְכָמוֹהוֹ סוֹעֵר,

וְכוֹס יַיִן חָמַר / מְשַׁכֵּר כָּל נוֹעֵר,

מְמַהֵר תְּנוּמָתוֹ / לַעֲדֵי-עַד שְׁנָתוֹ!

אַשְׁרֵי אֱנוֹשׁ יִרְאֶה / בְאִישׁוֹנֵי מְזִמָּה,

כִּי גָאֹה תִגְאֶה / בְצֵאתָהּ הַנְּשָׁמָה,

וְתַעַל וְתִשְׁתָּאֶה / לְהֵיכְלֵי הַחָכְמָה,

וְתָתוּר צִדְקָתוֹ / מְנוּחָה לַחֲנוֹתוֹ.
