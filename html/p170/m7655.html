[קוּמִי בַת וּלְכִי]

משה אבן עזרא

סימן: משה

קוּמִי בַת וּלְכִי / שׁוּבִי לְעָרַיִךְ / שְׁלִישִׁיָּה

הִנֵּה אָנֹכִי / שֹׁלֵחַ לָכֶם / אֶת אֵלִיָּה: [מלא' ג, כג] מֵאַהֲבָה חוֹלָה / יוֹנָתִי בְּחַגְוֵי סֶלַע / נָדָדָה

אֵין מוֹשִׁיעַ לָהּ / צָרָה כְּמַבְכִּירָה / יוֹלֵדָה: [ירמ' ד, לא]

      נוֹהֶמֶת בִּבְכִי

      תּוֹךְ מִכְמָר

      חֵלֶף זְמָר

      מִבּוֹר תַּחְתִּיָּה [אי' ג, נה]

הִנֵּה אָנֹכִי / שֹׁלֵחַ לָכֶם / אֶת אֵלִיָּה: שֵׁשׁ מֶשִׁי רִקְמָה / תַּעְדִּי עֲדִי פָז / לְבוּשׁ סַהֲרוֹן,

טוּרֵי אַחְלָמָה / אֶל צַוְרֹנַיִךְ / יְהִי יִתְרוֹן

וּכְאוֹר תַּעֲלֻמָה / הוֹד תִּלְבְּשִׁי אַבְנֵי / זִכָּרֹן: [שמ' כה, יב]

      בַּת נָדִיב עִרְכִי

      שִׁיר מַהֲלָל

      כִּפְרֹט כְּלָל

      לִשְׁלֹל שְׁלָל

      עוֹד בִּי מַשְׂכִּיָה [יחז' לח, יג]

הִנֵּה אָנֹכִי / שֹׁלֵחַ לָכֶם / אֶת אֵלִיָּה: הֵילֵל בֶּן שַׁחַר / יִנּוֹן שְׁמוֹ, הַכֹּל נִכְסָפִים

כִּי אוֹתוֹ בָחַר / צוּר לִפְנֵי שֶׁמֶשׁ / וּשְׂרָפִים –

מַה דּוֹדִי אֵחַר / אֶת פַּעֲמֵי רִכְבּוֹ / הַיָּפִים: [שופ' ה, כח, שיה"ש ז, ב]

      הֶמֵּאִיר חָשְׁכִּי

      תִּקְרָא דְרוֹר

      וּצְרֹר צְרוֹר

      וּבְרֹר בְּרֹר

      יוֹשְׁבֵי מַאְפֵּלְיָה [ ירמ' ב, לא]

הִנֵּה אָנֹכִי / שֹׁלֵחַ לָכֶם / אֶת אֵלִיָּה:
