נוֹטֵה עֲלִיָּה

נסים מצליח

סימן: נשים. מנגינה: צבא

נוֹטֵה עֲלִיָּה / חוּס רַחֲמֵנִי

חַי, חַי, נָא הַחְלִימֵנִי / וְגַם הַחְיֵינִי

צֶדֶק לְעַבְדָּךְ / עֻזִּי וְאוֹנִי

שֶׁבַח אַהֲבָה

מַרְבֶּה זְמִיר יָהּ

חַי, לָךְ שָׁב אֲנִי

חָנֵּנִי וַעְנֵנִי.

שִׂימִי לְבִיָּא / לִבֵּךְ לְמוּלִי

חַי, חַי, אָז אֱלֵי מְעוֹנִי / אֶקְבֹּץ קְהָלִי

אָז לִי עֶבֶד דָּךְ / יִנְאַם לְמַעְנִי

שֶׁבַח אַהֲבָה

יוֹסֵד נְשִׁיָּה / קַבֵּץ פְּזוּרִי

חַי, חַי, נָא הַעֲלֵנִי / אֶל עִיר פְּאֵרִי

צַדִּיק בְּחַסְדָּךְ / שַׂגֵּב יְמִינִי

שֶׁבַח אַהֲבָה

מַהֵר אֵלִיָּה / לִפְנֵי מְשִׁיחִי

חַי, חַי, שְׁלַח וַאֲנִי / אֶשְׁעֶה נִיחוֹחִי

צַדִּיק עֶבֶד דָּךְ / יָשִׁיר רְנָנִי

שֶׁבַח אַהֲבָה.
