נְגִינָתִי אַזְכִּיר

נסים מצליח

סימן: נשׂים חזק

נְגִינָתִי / אַזְכִּיר לַיְלָה / יָהּ, עֲנֵנִי / פְּנֵה אֵלַי

זְמִירָתִי / יָהּ, שְׁעֵה לָהּ. / בָּרְכֵנִי / כְּמֵי אוּלָי.

שַׂמַּח לִבִּי / בְּנִיב חוֹזֶה / מַנֵּה מוֹנֶה / צֹאנְךָ יִמְנֶה

שׁוּבִי שׁוּבִי / וְנֶחֱזֶה. / אָנָּא מוֹנֶה / צֵא נָא שׂוֹנֵא.

חַזֵּק אוֹתִי / מֶלֶךְ עוֹלָם

מִשַּׁוְעָתִי / אַל תִּתְעַלָּם.
