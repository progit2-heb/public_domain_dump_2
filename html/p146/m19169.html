שמוט מוט רע

שלום שבזי

חתימה: שלם יוסף [שבזי]

שְׁמוֹט מוֹט רָע / שְׁהוּא נִבְהָל וְנִגְרָע / וְהָבִינָה לְכָל דִּבְרֵי לְשׁוֹנִי

לְבַשׁ מִרְבַּד וּבַד וּצְלַח בְּעֵץ לַח / וְאַל תֶּאְלַח / לְמוּל סוֹד הַצְּפוֹנִי

מְרֹאשׁ רֹאשׁ / צָר שְׁהוּא שׁוֹתֵק כְּחֵרֵשׁ / וּמִתְגָּרֵשׁ / בְּתוֹךְ הַדְרַת צְפוּנִי

יְעַקֵּשׁ גַּם יְנַקֵּשׁ / לָךְ יְבַקֵּשׁ / לְהַסִּיתָךְ בְּעִרְבּוּב רַעֲיוֹנִי

וְעַתָּה לֵךְ שְׁלֵךְ מִלָּיו וְהַחְלֵץ / לְעֵצוֹתָיו וְקָרֵב קֵץ יְמִינִי

סְבַר בַּר לֵב / וְלַחָכְמָה תְשַׁלֵּב / וְזֶה דַעְתִּי וְזֶה חֵפֶץ רְצוֹנִי

פְּלוֹט לוֹט וַעֲלוֹט / מִלָּיו וְהָלֵט / וּבוֹא אֶל תּוֹךְ מְכוֹנִי

וְשַׁר יוֹשֶׁר / וּבַשָּׁלוֹם תְּאֻשַּׁר / וְדַע פֵּשֶׁר בְּחָכְמַת תַּחְכְּמוֹנִי.
