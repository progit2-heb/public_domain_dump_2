צְפִירַת אֲהוּבוֹ

צאלח מצליח

זה השיר אשר שלח להיקר שי׳ צאלח דוד יעקב שנת התקט״ו

סימן: אני צאלח, ציון

צְפִירַת אֲהוּבוֹ / לְרֹאשׁ וַחֲלִי בוֹ / לְשׁוֹאֵל אֲשִׁיבוֹ / שְׁמִי הוּא בְקִרְבּוֹ

צְבָא בִין וְחֵילוֹ / בְּלִבּוֹ שְׁבִילוֹ / וְחָכָם בְּשִׂכְלוֹ / לְאָחוֹר מְשִׁיבוֹ

יְקוֹד אֵשׁ שְׁלוֹמִי / תְּלַהֵב בְּגָלְמִי / אֲבָל בַּחֲלוֹמִי / אֲכַבֶּה שְׁבִיבוֹ

וְלוּ אוֹן בְּיָדִי / לְעוֹפֵף לְדוֹדִי / וְאַשְׁקִיט יְקוֹדִי / בְנָאְמוֹ וְנִיבוֹ

נְדוּדַי בְּשׂוּם בַּ/עֲצָמַי כְּלַבָּה / וְתִשְׁקַע בְּעֵת בָּא / לְיָדִי כְתָבוֹ

לְךָ נָא כְתָבִי / לְיַד אִישׁ חֲבִיבִי / וְתַזִּיל רְבִיבִי / וְטַל עַל אֲבִיבוֹ.
