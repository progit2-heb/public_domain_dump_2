[אֲדּוֹן הַכֹּל וְחַי עוֹלָם]

מרדכי דאטו

ں – – –

אֲדּוֹן הַכֹּל /וְחַי עוֹלָם / וְעַד אֵין סוֹף / תְּחִלָּתוֹ

בְּכֶתֶר אוֹר / וְהוּא נֶעְלָם / בְּחָכְמָתוֹ / וּבִינָתוֹ

רְצוֹן חַסְדוֹ / לְאַבְרָהָם / וּמִי כָאֵל / גְּדֻלָּתוֹ

הֲלִיכוֹתָיו / בְּמִדּוֹתָיו / וְעַל יִצְחָק / גְּבוּרָתוֹ

מְקַבֵּל הָ־ / אֱמֶת יַעֲקֹב / וְלַהְקִימוֹ / בְתִפְאַרְתּוֹ

מְקוֹם נֶצַח / וְהוֹד צַדִּיק / יְסוֹד עוֹלָם / וּמַלְכוּתוֹ

יְהִי רָצוֹן / לְאִמְרֵי פִי / וְהֶגְיוֹן לֵב / וְיִרְאָתוֹ

כְּמוֹ חֶרֶב / בְּפִיפִיּוֹת / לְצַר צוֹרֵר / לְהַכְרִיתוֹ

אֱמוּנָתִי / יְחִידָתִי / תְּנִי לָאֵל / וְאַחְדּוּתוֹ

לְמָגֵן לִי / כְאַבְרָהָם / רְדֹף חַסְדוֹ / כְּמִדָתוֹ.
