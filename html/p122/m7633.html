ביום הבכורים, בהקריב ספר התורה מנחה חדשה לי"י

אפרים לוצטו

קוֹל שִׁגָּיוֹן / אֶל אֵל עֶלְיוֹן,

כִּי מִצִּיּוֹן / תֵּצֵא תוֹרָה:

קוֹל הַקִּרְיָה, / כִּי הוֹמִיָה,

וּתְהַלֵּל יָה / עַל הַתּוֹרָה: עַל הַר אֵיתָן / כִּדְמוּת חָתָן

יוֹם זֶה נָתַן / אֶת הַתּוֹרָה: יוֹם זֶה רוֹמָם / עַל כָּל-יוֹמָם,

כִּי תִתַּמָּם / בּוֹ הַתּוֹרָה: יוֹם זֶה רִאשׁוֹן / אֶל כָּל-שָׂשׂוֹן,

כָּל-אִישׁ לָשׁוֹן / יַגְדִּיל תּוֹרָה: אַף בִּרְעָדָה / כָּל-הָעֵדָה

תִתֵּן תּוֹדָה / אֶל הַתּוֹרָה: יוֹם זֶה אַשְׁרֵי / אִישׁ הַיָּרֵא

אֶת כָּל-דִּבְרֵי / זֹאת הַתּוֹרָה: כִּי כָאֶזְרַח / יָצִיץ יִפְרַח,

עָלָיו יִזְרַח / אוֹר הַתּוֹרָה: יוֹם זֶה יוֹאֵל / יָקִים הָאֵל

אֶת הַהַרְאֵל / כִּדְבַר תּוֹרָה: יוֹם זֶה יָעִיר / נִקְמַת הָעִיר

בִּבְנֵי שֵׂעִיר / בּוֹזֵי תוֹרָה:

צָרָיו יַכְלִים, / כִּסְאוֹ יַשְׁלִים,

קוֹל הִלּוּלִים / תִּשָּׂא תוֹרָה: קוֹל שִׁגָּיוֹן / אֶל אֵל עֶלְיוֹן,

כִּי מִצִּיּוֹן / תֵּצֵא תוֹרָה.
