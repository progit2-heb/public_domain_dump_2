[שְׂעִפַּי יָנוּעוּ]

וידל בנבנשתי

כתובת: עוד לו

שְׂעִפַּי יָנוּעוּ / וְכַיּוֹם נָסָעוּ / לְבַקֵּשׁ אֶת דּוֹדָם

וְאַחְרָיו יִנָּהוּ / לְשָׁרְתוֹ כִי זֶה הוּא / יְקָרָם וּכְבוֹדָם

כְּרוּב יָדוֹ הָדָה / בְתוֹרָה וּתְעוּדָה / וְיוֹדֵעַ סוֹדָם

יְאַסֵּף כָּל יָקָר / וְחַכְמֵי הַמֶּחְקָר / בְּרוּחוֹ הֶעְמִידָם

הֲיָמוּתוּ רוֹאָיו / וְשׁוֹמְעֵי חִין שִׁירָיו / כְּמוֹת כָּל הָאָדָם
