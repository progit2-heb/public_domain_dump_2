[מוֹשִׁיעִי לְעִתּוֹת צָרָה גְּאֻלַּת עַמְּךָ הוֹאֵל] (גאולה)

משולם הסופר מפרוג'ה

[מפֵּרוּגָ֗ה (Perugia) בפלך טוֹסְקָנָה, חי כנראה במחצית הראשונה של המאה הי"ד]

גְּאֻלָּה

מוֹשִׁיעִי לְעִתּוֹת צָרָה / גְּאֻלַּת עַמְּךָ הוֹאֵל

אָנָּא שִׂים לָחֳרָבוֹת בָּצְרָה / לְשַׁמָּה קֵדָר וְאַרְבֵאל.

מְרוֹם שִׁבְתּוֹ שׁוֹכֵן סֶלַע / גַּאֲוַת לִבּוֹ הַשְׁפִּילֵהוּ

רֹבֶה קַשָּׁת תִּבְלַע / תִּפְתַּח־אֶרֶץ תְּכַסֵּהוּ

וְלוֹבֵשׁ רִקְמָה וְתוֹלָע / גֶּבֶר עֲמִיתְךָ כַּלֵּהוּ

וְתוֹרִיד לָאָרֶץ נִצְחָם / וּרְמֹס בְּחֵמָה מַגְדִּיאֵל

וְכָל־מַאֲמַצֵּי כֹחָם / וּבַל־יִזָּכֵר יִשְׁמָעֵאל.

שַׁוְעַת עַמְּךָ אָבִי זְכֹר / מַהֵר שְׁנַת פְּקֻדָּתָם

וּבְרִית בְּחִירֶיךָ תִּזְכֹּר / תּוֹשִׁיעֵם בַּעֲבוּר אַהֲבָתָם

יְדִידֶיךָ לָמָּה תִמְכֹּר / שְׁעֵה־נָא אֶל־תְּפִלָּתָם

חֹן עַל־פְּלֵיטַת עַמֶּךָ / וְרַחֶם־נָא לְדַל שׁוֹאֵל

וְכַפֵּר לְמַעַן שְׁמֶךָ / כִּי תְּעוּדָה הִיא בְּיִשְׂרָאֵל.

לִבְשִׁי עֻזְּךָ בִּזְרוֹעֶךָ / עוּרִי עוּרִי כִּימֵי קֶדֶם

זִכְרוֹנְךָ וְשִׁמְעֶךָ / לֹא תִּשְׁכַּח יָמִים מִקֶּדֶם

וְכַנָּה אֲשֶׁר הִיא נִטְעֶךָ / תְּכִינֶהָ כְמִקֶּדֶם

בָּרֵךְ חֵילְךָ וְלֹא תִקֹּב / כִּי מָגִנֵּנוּ רַחֲמַי אַל

הַבָּאִים יַשְׁרֵשׁ יַעֲקֹב / יָצִיץ וּפָרַח יִשְׂרָאֵל.

מֵקִים אֶת־דִּבְרֵי עֲבָדָיו / וַעֲצַת מַלְאָכָיו יַשְׁלִים

וְאוֹהֵב צֶדֶק לִידִידָיו / וְשׁוֹמֵעַ צַעֲקַת דַּלִּים

מְרַחֵם לְמַעֲשֵׂה יָדָיו / יַגְבִּיהַּ קֶרֶן שְׁפָלִים

חוּשָׁה לִפְדּוֹת אֶת־עָם נוֹשַׁע / עֵת יַעֲמֹד הַשַּׂר מִיכָאֵל

וּשְׂרִידַי עַמְּךָ הוֹשַׁע / וְתָשִׁיב גּוֹלַת אֲרִיאֵל.
