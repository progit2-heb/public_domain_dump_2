אֵלֶיךָ שִׁוַּעְתִּי

משה חוצין

סימן: א״ב. בראש הצלעיות האחרונות בכל טור: משה חוצין. מנגינה: באבא טאהר. לחן: יאמר נא ישראל.

אֵלֶיךָ שִׁוַּעְתִּי / בְּשִׁמְךָ בָטַחְתִּי / גָּדְלְךָ סִפַּרְתִּי / מְהֻלָּל יְיָ

דַּלְתְּךָ שָׁקַדְתִּי / הַקְשִׁיבָה רִנָּתִי / וְגַם לִתְפִלָּתִי / שְׁמָעָה יְיָ

זְכֹר נָא בְרִיתִי / חוּשָׁה לְעֶזְרָתִי / טֶרֶם לָךְ קָרָאתִי / הוֹשִׁיעָה יְיָ

יִשְׁעֲךָ צִפִּיתִי / כּוֹנֵן בֵּית תִּפְאַרְתִּי / לְהַקְרִיב מִנְחָתִי / חֲדָשָׁה לַיְיָ

מִשְׂגָּב בְּצָרָתִי / נְקֹם אֶת נִקְמָתִי / סַלְעִי וּמְצוּדָתִי / וּמְנוּסִי יְיָ

עֲנֵה עֲתִירָתִי / פְּנֵה לִתְחִנָּתִי / צַוֵּה יְשׁוּעָתִי / צוּר יִשְׁעִי יְיָ

קָרֵב קֵץ פְּדוּתִי / רַחֵם עַל נִשְׁמָתִי / שְׁמֶךָ שָׁמַעְתִּי / יָרֵאתִי יְיָ

תָּאִיר אֲפֵלָתִי / תָּחִישׁ גְּאֻלָּתִי / תָּשִׁיב נַחֲלָתִי / נְחֵנִי יְיָ.
