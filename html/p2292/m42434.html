אֵל יְבָרֵךְ

סלימאן מנחם מני

שלחתי לחתן אביו יוסף

אֵל יְבָרֵךְ / חָתָן אַבְרֵךְ / שַׂר אֵלִיָּהוּ / בְּכָל טוֹבָה

וּלְעֻמָּתוֹ / נְוַת בֵּיתוֹ / צְבִיַּת חֵן / מַאנְיָה חֲבִיבָה

עֲלֵי אֹשֶׁר / וְגַם עֹשֶׁר / הָרֵם דִּגְלָם / בְּכָל חִבָּה

וּבְרֹב בָּנִים / וּשְׂשׂוֹנִים / עַל בִּרְכֵּי יוֹסֵף / יְנוּבוּן בְּשֵׂיבָה.
