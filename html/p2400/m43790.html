אֵל חַי

יוסף חיים מבגדאד

לכבוד משיח בן דוד. סימן: אהבה נצר לנו

ב

אֵל חַי מַעְלָה / לְךָ אוֹחִילָה / מִמְּךָ אֶשְׁאֲלָה / תֶּן-לִי נַחֲלָה

בְּבֶן יִשָׁי.

הַשְׁמֵד הַזָּרִים / אוֹיְבִים וְצָרִים / נָכְרִים הַצּוֹרְרִים / אֲשֶׁר הֵם אוֹמְרִים

מִי בֶן יִשָּׁי.

בְּנֵה אֲרִיאֵל / לְמַעַנְךָ אֵל / וּשְׁלַח נָא הַגּוֹאֵל / מֶלֶךְ יִשְׂרָאֵל

בֵּן לְיִשָׁי.

הָשֵׁב עֲטָרָה / עַל רֹאשׁ גְּבִירָה / וְהַזְרַח נָא אוֹרָה / חֹטֶר תִּפְאָרָה

גֶזַע יִשָׁי.

נָא הוֹשִׁיעָה נָּא / וְהַצְלִיחָה נָּא / כִּי עֵת לְחֶנְנָהּ / יָבוֹא בְרִנָּה

שֹׁרֶשׁ יִשָׁי. צֶמַח הִצְמַחְתָּ / כִּסְאוֹ כוֹנַנְתָּ / וְאוֹתוֹ אֲהַבְתָּ / כִּי בוֹחֵר אַתָּה

בְּבֶן יִשָׁי.

רַגְלֵי הַמְבַשֵּׂר / שָׁלוֹם וָאֹשֶׁר / הֵם קַלּוּ מִנֶּשֶׁר / אוֹתִי יְבַשֵּׂר

בָּא בֶּן יִשָׁי.

לוֹ יְבַקְּשׁוּ / אוֹתוֹ יִדְרְשׁוּ / אוֹיְבָיו יֵבוֹשׁוּ / אֵלָיו יַגִּישׁוּ

מְלָכִים שָׁי.

נָכוֹן הוּא כִּסְאוֹ / אֵל חַי נְשָׂאוֹ / כָּל אַנְשֵׁי צְבָאוֹ / דִּרְשׁוּ יוֹם בּוֹאוֹ

מֵאֵל עוֹשָׂי.

וְחִישׁ נַקְרִיבָה / בְּשִׂמְחָה רַבָּה / וּבְרֹב אַהֲבָה / שַׁלְמֵי נְדָבָה

לֶחֶם אִשַּׁי.
