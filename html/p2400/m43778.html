עֲלִי בְאֵר

יוסף חיים מבגדאד

סימן: אבחיכמנצשת

עֲלִי בְאֵר, עֲלִי בְאֵר, / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

אֲגַם מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

אוֹצַר מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

בְּאֵר מַים מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

חָבִית מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

יְאוֹר מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

כְּלִי מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

כּוֹס הַמַּיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

מְקוֹר מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

מַעְיַן מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

מִקְוֵה מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

מוֹצָא מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

נְהַר מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

צִנּוֹר מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

שְׁבִיל מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי

תְּהוֹם מַיִם מָלֵא מַיִם / פַּלְגֵי מַיִם, פַּלְגֵי מַיִם,

עֲלִי
