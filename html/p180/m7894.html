שלמה אבן גבירול

[חסם את עדיך] / שלמה אבן גבירול

ולה פי מן אחתקרה [ולו על מי שזלזל בו]

È — — — / È — — — / / È — — — /È — — —

חֲסֹם אֶת עֶדְיְךָ וּבְלֹם, / וְאֵל יִגְעַר בְּךָ, שָׂטָן!

נְקַלּוֹתִי בְּעֵינֶיךָ, / לְמַעַן כִּי אֲנִי קָטָן:

חֲקֹר מִלַּי וְתֵדַע כִּי / אֲנִי עֵבֶר אֲבִי יָקְטָן!

את הטקסט לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט. הוא זמין תמיד בכתובת הבאה: https://benyehuda.org/read/7894
