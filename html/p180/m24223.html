יְיָ שֵׁם אָיֹם

שלמה אבן גבירול

מִי יְמַלֵּל גְּבוּרוֹת יְיָ

יְיָ שֵׁם אָיֹם וְנוֹרָא / שֶׁבּוֹ עוֹלָם נִבְרָא

שֶׁכֵּן בִּהְיוֹתוֹ נִקְרָא / הַשָּׁמַיִם שָׁמַיִם לַיְְיָ

יְיָ לְשִׁבְעָה שֵׁמוֹת מְפֹרָשׁ / לְמִדַּת רַחֲמִים נִדְרָשׁ

לָכֵן בַּעֲבֹר כְּבוֹדוֹ אָרַשׁ / וַיִּקְרָא יְיָ

יְיָ מְחֻלָּק לְשִׁשָּׁה פָנִים / מוֹסְדֵי שִׁשָּׁה דְפָנִים

מוּל אֶל מוּל מְכֻוָּנִים / מָה רַבּוּ מַעֲשֶׂיךָ יְיָ

יְיָ הֻפְרַד לַחֲמִשָּׁה חֲצָיִים / הֵם בִּכְתָב וְלָשׁוֹן מְנוּיִים

הוֹדוֹ יְהַלְלוּ דָרֵי נְטוּיִים / וְיוֹדוּ שָׁמַיִם פִּלְאֲךָ יְיָ

יְיָ הוּחַק בְּאַרְבָּעָה אוֹתוֹת / הֵן לְאַרְבָּעָה שֵׁמוֹת חֲרוּתוֹת

הִנָּם כְּרוּתוֹת וַעֲמוּתוֹת / רַבּוֹת עָשִׂיתָ אַתָּה יְיָ

יְיָ קִבֵּץ אוֹתוֹת שְׁלֹשָׁה / קָצַב שֵׁשׁ כְּאַחַת פְּרוּשָׁה

קְדֻשָּׁתוֹ בִּשְׁלֹשָׁה אֲרוּשָׁה / קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ

יְיָ טִכֵּס שֵׁם רִאשׁוֹן לִשְׁנַיִם / טָעַם יָהּ לְהָאִיר עֵינַיִם

טְפוּחִים בּוֹ שָׁקַל בְּמֹאזְנַיִם / כִּי בְּיָהּ יְיָ

יְיָ נִשְׂגָּב בְּאֶחָד וּמִי יְשִׁיבֶנּוּ / נִמְצָא בַכֹּל וּמִי יְסִבֶּנּוּ

נִקְשָׁר בַּלֵּב וְלֹא יַעַזְבֶנּוּ / כָּל עַצְמוֹתַי תֹּאמַרְנָה יְיָ.
