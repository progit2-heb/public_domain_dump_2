שׁוֹאֵף כְּמוֹ עֶבֶד

שלמה אבן גבירול

שׁוֹאֵף כְּמוֹ עֶבֶד / יִשְׁאַף לְיַד רַבּוֹ / חַסְדָּךְ פְּרֹשׂ עָלָיו / וּקְרַע שְׁטַר חוֹבוֹ

לוּלֵי אֲמִירָתְךָ / שׁוּבוּ וְאָשׁוּבָה / הָיָה כְּרַב חוֹבֵל / נִכְנַע בְּמַצָּבוֹ

מַנְעוּל תְּשׁוּבָתָךְ / תִּפְתַּח לְעֶבֶד קָם / טֶרֶם לְהִתְפַּלֵּל / לָךְ, יַעֲרֹך נִיבוֹ

הָהּ לוֹ בְעֵת רוֹאוֹ / שֵׂיבָה זְרוּקָה בוֹ / לִבּוֹ בְיַם חֶטְאוֹ / יַשְׂחֶה בְמֵי עָצְבּוֹ

נִשְׁמַת מְיַחֶלְךָ / תָּעִיד עֲלֵי לִבּוֹ / לֹא יַעֲשֶׂה עָוֶל / מֵאַחֲרֵי שׁוּבוֹ.
