[הלנצח יעירני ולעד]

שלמה אבן גבירול

וקאל פי אלתגז ّ ל [ואמר בענין האהבה]

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

הֲלָנֶצַח יְעִירֵנִי וְלָעַד / יְעַרְעֵר מוֹסְדֵי-חֵשֶׁק בְּשַׁלְוִי

אֲשֶׁר פָּנָיו פְּנֵי יוֹסֵף בְּיָפְיוֹ / אֲבָל כֵּלָיו כְּלֵי שִׁמְעוֹן וְלֵוִי

וְלוּ נַפְשִׁי לְקָחָה לוֹ אֲזַי שָׁב / לְבַקֵּשׁ עוֹד שְׁאָר נַפְשִׁי בְּגֵוִי.
