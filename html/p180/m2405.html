[הֲיִחְיֶה אִישׁ]

שלמה אבן גבירול

וקאל איצֹא

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

הֲיִחְיֶה אִישׁ אֲשֶׁר נָסַע יְדִידוֹ / וְשַׂר הַחֵן וְהַיֹּפִי רְבִידוֹ

אֲשֶׁר הִפְקִיד לְבָבוֹ עִם-לְבָבִי / וְשַׂמְתִּי כֵן אֲנִי לִבִּי בְיָדוֹ

וְשִׁלַּחְתִּיו וְלִבִּי בּוֹ מְקֻשָּׁר / כְּלֵב יִצְחָק בְּעֵת אָבִיו עֲקָדוֹ.
