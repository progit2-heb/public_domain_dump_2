


	
	שובי יפיפיה חלי פני דודך
	



חתימה: שלם [שבזי] ש״צ [שמרו צורו]




שובי יפיפיה / חלי פני דודך



אל תישני רעיה / רב לך צפון סודך



עצה ותושיה / בם חזקי ידך



אז תמצאי מחיה / אם יגברה חסדך



שובי ברון / הסהרון



לבשי לחן / ועשי רצון



האל שהוא קונך



לו תשאלי / ירפא חלי



אז תצהלי / גם תעלי



תמה לארמונך



מה טוב עלות / המגדלות



שם מגבלות / כל מקהלות



עמך והמונך



דוד שלחנו ערך / יגדל ויתברך



לבשי לבוש הדר / כי בא זמן ישעך



זיו בת אדום קדר / כי הרבתה מרעך



הודי לאל נאדר / מוחה לכל פשעך



עול הזמן נעדר / ויאתה שפעך



שפעת כבוד / נצח והוד



גם היסוד / כל אח ודוד



לך שופעים כלם



לא תיראי / המלאי



אל תדכאי / חן תמצאי



לפני אדון עולם



מיום זמן / כלכל במן



עם נאמן / ויקוממן



ויסבבה דגלם



חזק לכל מרך / זנם בכל צרך



משך זמן עלי / החן והטובה



ארצי ומגדלי / נשלל במעצבה



פשעי ומעללי / גרם והחובה



ואסבלה, אולי / אשמע בדודי בא



שועי שעה / צורי דעה



צאנך רעה / גם הסעו



על מחמד הרים



לבני סגול / זכרה גמול



עליו חמול / וביום המול



אל ימשלו זרים



מלכך שלח / עם נאלח



דרכו צלח / אורו זרח



ילך במישרים



גם חזקה ברך / עם נעבד פרך



שור משמי מרום / וראה עני עמך



דגלו בחן הרם / קנא אדון לשמך



ורצה והחזירם / לראות במנעמך



ומחל לחטא גורם / ומרחקם ממך



שוב אל מקום / קדשך, נקם



מדוחקם / יד צר עקם



ושפט להר עשו



לו שלמה / אף וחמה



אז ייהמה / אל ישטמה



עמך ויכעיסיו



מחסדך / צו שפעך



אל עמך / תן ישעך



אז יבשו שוסיו



כי הזמן ארך / עלי ולבי רך



צורי ומגני / עזרת בני איש תם



אל תרחקה מני / גלה לאור נעתם



אשוב מקום גני / מקץ אשר נסתם



ותשמעה אזני / שמץ ולא נחתם



מה יענה / לב נענה



לא יהנה / במחנה



מזיו כבוד מעלה



שמם וסר / עלי כשר



ויחסר / במאסר



עם בעלי עולה



תמה מאד / ויחמד



לידע בסוד / ויעמד



זכר עני חלה



אלי ראה / צורי דעה



שועי שעה / אותי פדה



מיד בני עולה



בא צד שמאל ערך / ללחם וחץ דרך



אחתם שבח אל חי / בלשון מצחצחת



לשון כמלקוחי / לשמו משבחת



שלום לכל אחי / תזכיר ושומחת



ובסוד להפציחי / תמיד מפענחת



מלכי מחול / עון כחול



עמך נחול / טבעו שחול



נשמע בבוא תשבי



שועת עלוב / שמעה ושוב



רפא משוב / לבו נבוב



הסר לכל עצבי



תקשיב לקול / דל מנפול



חסד גמול / ימיו כצל



וימינך תצביא



אללה וחיד / איצא וסיד



ירבל עביד / יסלי אלנכיד



ואתוב עין דנבי



[האל יחיד / וגם אדון



מקבל תשובת יראים / מנחם נדכאים



אחזר בתשובה מחטאי]



כי אין לך ערך / תגדל ותתברך.
